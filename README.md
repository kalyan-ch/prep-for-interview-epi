## Primitive Types

* number of digits in an integer x is n = Math.floor(Math.log10(x)) + 1
* is n a power of 2 (n & (n-1) == 0)
* Right propagate last set bit:  (num | (num -1))
* divide by 2 = n >>> 1
* multiply by 2 = n << 1
